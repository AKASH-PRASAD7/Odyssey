# Cheat Sheet: Building Unsupervised Learning Models

## Unsupervised Learning Models

| Model Name             | Brief Description                                                                                                                                                                                                                                                              | Ordered Code Syntax                                                                                                                                                                   |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **UMAP**               | **Uniform Manifold Approximation and Projection** is used for dimensionality reduction. <br><br> **Pros:** High performance, preserves global structure. <br> **Cons:** Sensitive to parameters. <br> **Applications:** Data visualization, feature extraction.                | **1. Import UMAP**  <br> `from umap.umap_ import UMAP\n` <br><br> **2. Initialize Model** <br> `umap = UMAP(n_neighbors=15, min_dist=0.1, n_components=2)\n`          |
| **t-SNE**              | **t-Distributed Stochastic Neighbor Embedding** is a nonlinear dimensionality reduction technique. <br><br> **Pros:** Excellent for visualizing high-dimensional data. <br> **Cons:** Computationally expensive. <br> **Applications:** Data visualization, anomaly detection. | **1. Import TSNE** <br> `from sklearn.manifold import TSNE\n` <br><br> **2. Initialize Model** <br> `tsne = TSNE(n_components=2, perplexity=30, learning_rate=200)\n` |
| **PCA**                | **Principal Component Analysis** is used for linear dimensionality reduction. <br><br> **Pros:** Easy to interpret, reduces noise. <br> **Cons:** May lose nonlinear information. <br> **Applications:** Feature extraction, compression.                                      | **1. Import PCA** <br> `from sklearn.decomposition import PCA\n` <br><br> **2. Initialize Model** <br> `pca = PCA(n_components=2)\n`                                  |
| **DBSCAN**             | **Density-Based Spatial Clustering of Applications with Noise** is a density-based clustering algorithm. <br><br> **Pros:** Detects outliers. <br> **Cons:** Struggles with varying density clusters. <br> **Applications:** Anomaly detection, spatial clustering.            | **1. Import DBSCAN** <br> `from sklearn.cluster import DBSCAN\n` <br><br> **2. Initialize Model** <br> `dbscan = DBSCAN(eps=0.5, min_samples=5)\n`                    |
| **HDBSCAN**            | **Hierarchical DBSCAN** improves DBSCAN by handling varying density clusters. <br><br> **Pros:** Handles variable-density clusters better. <br> **Cons:** Slower than DBSCAN. <br> **Applications:** Large datasets, complex clustering.                                       | **1. Import HDBSCAN** <br> `import hdbscan\n` <br><br> **2. Initialize Model** <br> `clusterer = hdbscan.HDBSCAN(min_cluster_size=5)\n`                               |
| **K-Means Clustering** | **K-Means** groups data into *k* clusters using centroids. <br><br> **Pros:** Efficient and simple. <br> **Cons:** Sensitive to initialization. <br> **Applications:** Customer segmentation, pattern recognition.                                                             | **1. Import KMeans** <br> `from sklearn.cluster import KMeans\n` <br><br> **2. Initialize Model** <br> `kmeans = KMeans(n_clusters=3)\n`                              |

# Associated Functions Used

| Method                            | Brief Description                                             | Ordered Code Syntax                                                                                                                                                                                                                                                                                             |
| --------------------------------- | ------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **make_blobs**                    | Generates isotropic Gaussian blobs for clustering datasets.   | **1. Import Function** <br> `from sklearn.datasets import make_blobs\n` <br><br> **2. Generate Dataset** <br> `X, y = make_blobs(n_samples=100, centers=2, random_state=42)\n`                                                                                                                  |
| **multivariate_normal**           | Generates samples from a multivariate normal distribution.    | **1. Import Function** <br> `from numpy.random import multivariate_normal\n` <br><br> **2. Generate Samples** <br> `samples = multivariate_normal(mean=[0, 0], cov=[[1, 0], [0, 1]], size=100)\n`                                                                                               |
| **plotly.express.scatter_3d**     | Creates a 3D scatter plot using Plotly Express.               | **1. Import Plotly Express** <br> `import plotly.express as px\n` <br><br> **2. Create Plot** <br> `fig = px.scatter_3d(df, x='x', y='y', z='z')\nfig.show()\n`                                                                                                                                 |
| **geopandas.GeoDataFrame**        | Creates a GeoDataFrame from a Pandas DataFrame.               | **1. Import GeoPandas** <br> `import geopandas as gpd\n` <br><br> **2. Create GeoDataFrame** <br> `gdf = gpd.GeoDataFrame(df, geometry='geometry')\n`                                                                                                                                           |
| **geopandas.to_crs**              | Transforms the coordinate reference system of a GeoDataFrame. | **1. Convert CRS** <br> `gdf = gdf.to_crs(epsg=3857)\n`                                                                                                                                                                                                                                                 |
| **contextily.add_basemap**        | Adds a basemap to a GeoDataFrame plot.                        | **1. Import Contextily** <br> `import contextily as ctx\n` <br><br> **2. Plot GeoData** <br> `ax = gdf.plot(figsize=(10, 10))\n` <br><br> **3. Add Basemap** <br> `ctx.add_basemap(ax)\n`                                                                                               |
| **pca.explained_variance_ratio_** | Returns variance explained by each principal component.       | **1. Import PCA** <br> `from sklearn.decomposition import PCA\n` <br><br> **2. Initialize PCA** <br> `pca = PCA(n_components=2)\n` <br><br> **3. Fit Model** <br> `pca.fit(X)\n` <br><br> **4. Extract Variance Ratio** <br> `variance_ratio = pca.explained_variance_ratio_\n` |
